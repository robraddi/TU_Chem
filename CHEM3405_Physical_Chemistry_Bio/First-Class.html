2. a) Wave anatomy: Write down the mathematical form describing a sinusoidal wave $f(x, t)$, annotated with the wavelength $λ$, phase velocity $v$, and amplitude $A$.

\begin{align*} f(x,t=0) &= Acos(\frac{2\pi}{\lambda}(x-\nu t) + \phi) \\ &= Acos(k(x-\nu t) + \phi) \end{align*}

where $k = 2\pi/\lambda$

Do you recall how functions are manipulated?

import numpy as np

def function(x, t, A=1, nu=1, lam=1, phi=0):
